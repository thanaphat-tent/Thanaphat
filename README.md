# Project Setup Guide

## Tools Installed

This project utilizes the following tools and extensions:

- **Cursor IDE** - AI-powered code editor
- **Claude Code Add-on** - Claude AI integration for code assistance
- **Codex Add-on** - OpenAI Codex integration for code generation
- **VSCode** - Source code editor for code rewriting and checking

### Research & documentation

- **ChatGPT** - For research and finding experts with deep information
- **YouTube** - For transcripts of the experts
- **Glasp** - For detailed transcripts from the experts

## Setup Steps

Follow these steps to complete the project setup:

1. **Install Cursor IDE**
   - Download and install Cursor IDE from the official website
   - Complete the initial setup and configuration

2. **Install both Claude Code and Codex Add-on**
   - Install Claude Code Add-on in Cursor
   - Install Codex Add-on in Cursor
   - Configure API keys if required

3. **Use VSCode to create README.md file**
   - Open VSCode
   - Create or edit the README.md file with project documentation
   - Save the file

4. **Pushing the README.md file to Cursor**
   - Import or synchronize the README.md file to Cursor
   - Verify the file appears correctly in Cursor

5. **Push to GitHub**
   - Initialize a Git repository (if not already done)
   - Add the README.md file: `git add README.md`
   - Commit the changes: `git commit -m "Add README.md with project setup guide"`
   - Push to GitHub: `git push origin main`
